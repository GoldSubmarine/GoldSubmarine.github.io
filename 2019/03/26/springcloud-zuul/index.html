SpringCloud-Zuul

作者: GoldSubmarine 日期: 2019-03-26  17:33:45 分类: 软件技术

常见的网关服务有：Nginx，Tyk，Kong，Zuul。SpringCloud 的网关服务就是 Zuul。

Zuul 的并发性能并不高，但是功能强大，可以配合 Nginx 的高性能一起使用。

Zuul 的核心是一系列的过滤器，有四种 API 过滤器：

  • 前置(pre)，可用于限流，鉴权，参数校验
  • 路由(route)
  • 后置(post)，用于统计，日志
  • 错误(error)
