单线程

redis 是单线程的，一次只会执行一条命令，同时提交了多条指令，会把指令放到一个队列中，然后依次的执行。

redis 虽然是单线程，但是仍然高性能，因为：

  1. 纯内存（主要原因）
  2. 异步IO（使用了epoll）
  3. 避免线程切换消耗

因为是单线程，一条一条执行，所以要避免使用长（慢）命令，例如：keys, flushall, flushdb, slow lua script, mutil/ecex, operate big value(collection)
